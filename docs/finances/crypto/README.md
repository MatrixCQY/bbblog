# 加密货币与区块链 (Crypto & Blockchain)

## 1. 区块链基础 (Blockchain Basics)
- **分布式账本技术 (DLT)**
- **哈希函数与加密学**: SHA-256, 公钥/私钥, 椭圆曲线加密 (ECC)。
- **区块结构**: Merkle Tree, Block Header.

## 2. 比特币 (Bitcoin)
- **中本聪白皮书**: 点对点电子现金系统。
- **共识机制**: 工作量证明 (PoW)。
- **UTXO 模型**: 未花费交易输出。
- **挖矿与难度调整**。

## 3. 以太坊与智能合约 (Ethereum)
- **以太坊虚拟机 (EVM)**。
- **账户模型**: 外部账户 (EOA) vs 合约账户。
- **Gas 机制**: 防止死循环与资源滥用。
- **共识机制**: 权益证明 (PoS) - The Merge.
- **智能合约**: Solidity 编程基础。

## 4. 去中心化金融 (DeFi)
- **去中心化交易所 (DEX)**: AMM (Uniswap), Order Book.
- **借贷协议**: Aave, Compound.
- **稳定币**: USDT, USDC (法币抵押); DAI (加密资产抵押); 算法稳定币。

## 5. 前沿话题
- **Layer 2**: Rollups (Optimistic, ZK), Sidechains.
- **NFT**: 非同质化代币 (ERC-721).
- **Web3**: 去中心化互联网愿景。
