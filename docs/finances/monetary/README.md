# 货币金融学 (Monetary Economics)

## 1. 货币与银行 (Money & Banking)
- **货币的本质与职能**: 交易媒介、计价单位、价值储藏。
- **货币制度**: 金本位、信用货币。
- **商业银行**: 存款创造、资产负债表管理。

## 2. 中央银行与货币政策 (Central Banking)
- **中央银行职能**: 发行的银行、银行的银行、政府的银行。
- **货币政策工具**:
  - 公开市场操作 (OMO)
  - 存款准备金率
  - 再贴现率
- **货币政策目标**: 稳定物价、充分就业、经济增长、国际收支平衡。

## 3. 利率理论 (Interest Rates)
- **利率的决定**: 古典利率理论、流动性偏好理论、可贷资金理论。
- **利率结构**: 风险结构、期限结构 (收益率曲线)。

## 4. 通货膨胀 (Inflation)
- **成因**: 需求拉动、成本推动、结构性通胀。
- **菲利普斯曲线**: 失业与通胀的权衡。

## 5. 国际金融 (International Finance)
- **汇率制度**: 固定汇率 vs 浮动汇率。
- **国际收支**: 经常账户、资本账户。
- **蒙代尔-弗莱明模型 (Mundell-Fleming Model)**。
