# 市场结构 (Market Structures)

## 1. 完全竞争 (Perfect Competition)
### 特征
-   无数买者卖者。
-   同质产品。
-   自由进出。
-   信息完全。

### 均衡
-   **企业是价格接受者**: $P = MR$。
-   **短期均衡**: $P = MC$。可能盈利或亏损。
    -   停业点: $P < AVC$。
-   **长期均衡**: 自由进出导致利润为零，价格等于最小 ATC。$P = MC = \min(ATC)$。
-   **效率**: 达到配置效率 ($P=MC$) 和生产效率 (最小 ATC)。

## 2. 垄断 (Monopoly)
### 特征
-   只有一个卖者。
-   产品无替代品。
-   进入壁垒高（资源垄断、政府特许、自然垄断）。

### 均衡
-   **企业是价格制定者**: 面临向下倾斜的需求曲线。$MR < P$。
-   **最大化利润**: $MR = MC$。
-   **结果**: $P > MC$，产量低于完全竞争，存在**无谓损失 (Deadweight Loss)**。

### 价格歧视
-   **一级**: 向每个消费者索取其最高支付意愿（榨干消费者剩余）。
-   **二级**: 按数量定价。
-   **三级**: 按群体定价（如学生票）。

## 3. 垄断竞争 (Monopolistic Competition)
### 特征
-   许多卖者。
-   **产品差异化**（关键特征）。
-   自由进出。

### 均衡
-   短期类似垄断。
-   长期利润为零（需求曲线切于 ATC）。
-   **产能过剩**: 产量小于最小 ATC 对应产量。价格 > 边际成本。

## 4. 寡头 (Oligopoly)
### 特征
-   少数几个卖者。
-   相互依存（策略互动）。

### 模型
-   **博弈论 (Game Theory)**: 纳什均衡。囚徒困境说明为何卡特尔（Cartel）难以维持合作。
-   **古诺模型 (Cournot)**: 数量竞争。
-   **伯特兰模型 (Bertrand)**: 价格竞争。
-   **斯塔克伯格模型 (Stackelberg)**: 领导者-追随者模型。
