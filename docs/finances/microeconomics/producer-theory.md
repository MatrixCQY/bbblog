# 生产者理论 (Producer Theory)

## 1. 生产函数 (Production Function)
$$
Q = f(K, L)
$$
描述投入（资本 K，劳动 L）与产出 Q 之间的技术关系。

### 1.1 短期生产
资本 K 固定，只变动劳动 L。
-   **边际产量 (MPL)**: $\Delta Q / \Delta L$。
-   **边际产量递减规律**: 在其他投入不变时，连续增加某一种投入，新增产出最终会下降。

### 1.2 长期生产
所有要素均可变。
-   **等产量曲线 (Isoquant)**: 产出相同的 K, L 组合。
-   **边际技术替代率 (MRTS)**: $MRTS = MP_L / MP_K$。
-   **规模报酬 (Returns to Scale)**: 所有投入同比例增加：
    -   产出增加比例 > 投入比例 $\rightarrow$ 规模报酬递增 (IRS)。
    -   产出增加比例 = 投入比例 $\rightarrow$ 规模报酬不变 (CRS)。
    -   产出增加比例 < 投入比例 $\rightarrow$ 规模报酬递减 (DRS)。

## 2. 成本 (Costs)
### 2.1 短期成本
-   **固定成本 (FC)**: 不随产量变化（如厂房租金）。
-   **可变成本 (VC)**: 随产量变化（如原料、工资）。
-   **总成本 (TC)**: $FC + VC$。
-   **边际成本 (MC)**: $\Delta TC / \Delta Q$。生产多一单位产品的成本。
-   **平均总成本 (ATC)**: $TC / Q$。U型曲线。
-   **关系**: MC 穿过 ATC 和 AVC 的最低点。

### 2.2 长期成本
-   长期平均成本 (LRAC) 曲线是短期平均成本曲线的包络线。
-   **规模经济**: LRAC 随产量增加而下降。
-   **规模不经济**: LRAC 随产量增加而上升。

## 3. 利润最大化
企业目标：最大化利润 $\pi = TR - TC$。
**通用条件**:
$$
MR = MC
$$
-   **边际收益 (MR)**: 多卖一单位获得的收入。
-   **边际成本 (MC)**: 多造一单位付出的成本。
-   若 $MR > MC$，增产；若 $MR < MC$，减产。
