# 量化金融 (Quantitative Finance)

## 1. 时间序列分析 (Time Series)
- **平稳性**: 单位根检验 (ADF)。
- **ARIMA 模型**: 自回归积分滑动平均。
- **GARCH 模型**: 波动率建模。

## 2. 风险管理 (Risk Management)
- **VaR (Value at Risk)**: 在险价值。
  - 历史模拟法
  - 蒙特卡洛模拟法
  - 参数法 (方差-协方差)
- **Expected Shortfall (ES)**: 尾部风险。

## 3. 算法交易 (Algorithmic Trading)
- **均值回归策略 (Mean Reversion)**: 配对交易 (Pairs Trading)。
- **动量策略 (Momentum)**: 趋势跟踪。
- **高频交易 (HFT)**: 做市策略、套利策略。

## 4. 随机微积分 (Stochastic Calculus)
- **布朗运动 (Brownian Motion)**: 维纳过程。
- **伊藤引理 (Itô's Lemma)**: 随机微分方程的链式法则。
