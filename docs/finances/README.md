## 期权策略

### 垂直价差 (Vertical Spread)

垂直价差是指同时买入和卖出同一标的资产、同到期日但行权价不同的期权。

## 希腊字母 (Greeks)

$$
\Delta = \frac{\partial V}{\partial S}
$$

$$
\Gamma = \frac{\partial^2 V}{\partial S^2}
$$

$$
\Theta = \frac{\partial V}{\partial t}
$$
