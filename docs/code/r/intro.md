# R 语言简介

## 1. 什么是 R？
R 是一种用于统计计算和图形的编程语言和软件环境。
- 广泛用于统计分析、数据挖掘、生物信息学。

## 2. 环境安装
- [CRAN](https://cran.r-project.org/) 下载 R。
- [RStudio](https://posit.co/download/rstudio-desktop/) 是最流行的 IDE。

## 3. 基本语法
- 赋值: `<-`
  ```r
  x <- 5
  y <- "Hello"
  ```
- 算术运算: `+`, `-`, `*`, `/`, `^`, `%%` (取模)

## 4. 包管理
- 安装包: `install.packages("ggplot2")`
- 加载包: `library(ggplot2)`
