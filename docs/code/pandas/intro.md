# Pandas 简介

## 1. 什么是 Pandas？
Pandas (Python Data Analysis Library) 是基于 NumPy 的开源数据分析库。
- 提供高性能、易用的数据结构。

## 2. 安装与导入
```bash
pip install pandas
```
```python
import pandas as pd
import numpy as np
```

## 3. 核心特性
- 处理缺失数据 (`NaN`)
- 强大的索引和切片功能
- 灵活的分组 (`groupby`)
- 时间序列分析
