# SQL 基础教程

## 1. 什么是 SQL？
SQL (Structured Query Language) 是用于管理关系数据库管理系统 (RDBMS) 的标准语言。
- **RDBMS**: MySQL, PostgreSQL, SQLite, SQL Server, Oracle.

## 2. 数据库操作 (DDL)
- **CREATE DATABASE**: 创建数据库
- **DROP DATABASE**: 删除数据库
- **CREATE TABLE**: 创建表
- **ALTER TABLE**: 修改表结构
- **DROP TABLE**: 删除表

## 3. 数据类型
- **数值**: `INT`, `FLOAT`, `DECIMAL`
- **字符串**: `VARCHAR`, `TEXT`, `CHAR`
- **日期/时间**: `DATE`, `DATETIME`, `TIMESTAMP`
