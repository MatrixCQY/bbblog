# C++ 基础教程

## 1. C++ 简介
C++ 是一种静态类型的、编译式的、通用的、大小写敏感的、不规则的编程语言，支持过程化编程、面向对象编程和泛型编程。

## 2. 环境设置
- 编译器: GCC, Clang, MSVC
- IDE: Visual Studio, CLion, VS Code

## 3. 基本语法
### Hello World
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

### 数据类型
- 基本类型：`int`, `char`, `float`, `double`, `bool`, `void`
- 修饰符：`signed`, `unsigned`, `short`, `long`

### 变量与常量
- 变量声明与定义
- 常量：`#define` 预处理器，`const` 关键字

## 4. 运算符
- 算术运算符
- 关系运算符
- 逻辑运算符
- 位运算符

## 5. 流程控制
- 条件语句：`if`, `else`, `switch`
- 循环语句：`for`, `while`, `do...while`
