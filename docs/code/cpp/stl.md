# C++ 标准模板库 (STL)

## 1. 容器 (Containers)
### 序列容器
- `vector`: 动态数组
- `deque`: 双端队列
- `list`: 双向链表

### 关联容器
- `set`, `multiset`
- `map`, `multimap`

### 容器适配器
- `stack`
- `queue`
- `priority_queue`

## 2. 算法 (Algorithms)
- `sort`
- `find`
- `binary_search`
- `reverse`

## 3. 迭代器 (Iterators)
- 输入迭代器
- 输出迭代器
- 前向迭代器
- 双向迭代器
- 随机访问迭代器
