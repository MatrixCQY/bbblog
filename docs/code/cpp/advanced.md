# C++ 高级特性

## 1. 模板 (Templates)
泛型编程的基础。
- 函数模板
- 类模板

## 2. 异常处理 (Exception Handling)
- `try`
- `catch`
- `throw`

## 3. 智能指针 (Smart Pointers)
C++11 引入，用于自动管理内存。
- `unique_ptr`
- `shared_ptr`
- `weak_ptr`

## 4. 多线程 (Multithreading)
- `std::thread`
- `std::mutex`
- `std::future`
