# 数据结构与算法简介

## 1. 什么是数据结构？
数据结构（Data Structure）是计算机存储、组织数据的方式。
- **逻辑结构**：线性结构、非线性结构。
- **存储结构**：顺序存储、链式存储、索引存储、散列存储。

## 2. 什么是算法？
算法（Algorithm）是解决特定问题求解步骤的描述，在计算机中表现为指令的有限序列。
- **特性**：有穷性、确定性、可行性、输入、输出。

## 3. 复杂度分析
- **时间复杂度**：Big O Notation ($O(1), O(n), O(\log n)$)
- **空间复杂度**：算法执行过程中所需要的最大存储空间。
