# 树与图 (Tree & Graph)

## 1. 树 (Tree)
- **二叉树 (Binary Tree)**：每个节点最多两个子节点。
- **二叉搜索树 (BST)**：左 < 根 < 右。
- **平衡树 (AVL, Red-Black Tree)**：保持高度平衡，保证操作 $O(\log n)$。
- **堆 (Heap)**：大顶堆、小顶堆。

## 2. 图 (Graph)
- **表示**：邻接矩阵、邻接表。
- **遍历**：
  - 深度优先搜索 (DFS)
  - 广度优先搜索 (BFS)
- **最短路径**：Dijkstra, Bellman-Ford, Floyd-Warshall。
- **最小生成树**：Prim, Kruskal。
