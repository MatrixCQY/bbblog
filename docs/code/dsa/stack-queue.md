# 栈与队列 (Stack & Queue)

## 1. 栈 (Stack)
后进先出 (LIFO)。
- **操作**：Push (入栈), Pop (出栈), Peek (查看栈顶)。
- **应用**：函数调用栈、表达式求值、括号匹配。

## 2. 队列 (Queue)
先进先出 (FIFO)。
- **操作**：Enqueue (入队), Dequeue (出队)。
- **应用**：任务调度、广度优先搜索 (BFS)。

## 3. 特殊队列
- **双端队列 (Deque)**：两端均可入队出队。
- **优先队列 (Priority Queue)**：按优先级出队 (堆实现)。
