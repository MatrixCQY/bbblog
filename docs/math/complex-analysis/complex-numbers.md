# 复数与复变函数

## 1. 复数的概念
复数 $z = x + iy$，其中 $x, y \in \mathbb{R}$，$i^2 = -1$。
- **模**: $|z| = \sqrt{x^2 + y^2}$
- **幅角**: $\arg z$
- **指数形式**: $z = re^{i\theta}$

## 2. 复平面
复数集 $\mathbb{C}$ 与平面 $\mathbb{R}^2$ 一一对应。
- **扩充复平面**: $\mathbb{C} \cup \{\infty\}$ (Riemann 球面)

## 3. 复变函数
设 $D \subset \mathbb{C}$，若对任意 $z \in D$，都有唯一的 $w \in \mathbb{C}$ 与之对应，则称 $w = f(z)$ 为 $D$ 上的复变函数。
