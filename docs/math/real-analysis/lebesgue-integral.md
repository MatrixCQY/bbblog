# Lebesgue 积分

## 1. Riemann 积分的局限性
- 黎曼积分依赖于定义域的划分，对函数的光滑性要求较高。
- 极限与积分交换次序的条件较强。

## 2. Lebesgue 积分的定义
基于值域的划分。
- 非负简单函数的积分
- 非负可测函数的积分
- 一般可测函数的积分

## 3. 极限定理
- **单调收敛定理 (MCT)**
- **Fatou 引理**
- **控制收敛定理 (DCT)**

## 4. $L^p$ 空间
- Hölder 不等式
- Minkowski 不等式
- $L^p$ 空间的完备性
