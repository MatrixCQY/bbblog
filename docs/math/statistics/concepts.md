# 数理统计的基本概念

## 1. 总体与样本
- **总体 (Population)**: 研究对象的全体。
- **样本 (Sample)**: 从总体中抽取的一部分个体 $X_1, X_2, \dots, X_n$。
- **简单随机样本**: 独立同分布 (i.i.d.)。

## 2. 统计量 (Statistic)
不含未知参数的样本函数。
- **样本均值**: $\bar{X} = \frac{1}{n}\sum X_i$
- **样本方差**: $S^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2$
- **次序统计量**: $X_{(1)} \le X_{(2)} \le \dots \le X_{(n)}$

## 3. 抽样分布
- $\chi^2$ 分布
- $t$ 分布
- $F$ 分布
